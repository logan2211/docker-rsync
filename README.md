# rsync docker image

Very simple rsync server image

## Usage

Mount configuration in `/etc/rsync.d` (directory) or at `/etc/rsync.conf `(file)
and it will be automatically loaded.

Modify the `RSYNC_CONFIG` env to set a custom base config path.
